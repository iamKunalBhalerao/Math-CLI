## Tech Stack

**System:** Node JS

## Setup Locally

Clone the project

```bash
  git clone https://github.com/iamKunalBhalerao/Math-CLI.git
```

Go to the project directory

```bash
  cd math-cli
```

Install all the dependancies

```bash
  npm install
```

Give execute permission

```bash
  chmod +x app.js
```

Install Locally in your Device

```bash
  npm link
```


## Usages
